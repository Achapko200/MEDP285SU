🆘 Help & FAQs

How do I create a new project?

Navigate to the Editor page and start typing your code in the playground. Your code will be saved automatically.

How do I edit an existing project?

Go to the Projects page, find your project, and click the ✏️ Edit button. This will open the code in the editor for editing.

Can I delete a project?

Yes! On the Projects page, click the 🗑 Delete button on the project you want to remove. You will be asked to confirm before deletion.

What is the AI Debugger?

The AI Debugger analyzes your code in real-time and gives helpful tips or suggestions to improve your code as you type.

How do I toggle dark mode?

Click the 🌙 button in the navigation bar to switch between light and dark modes for better viewing comfort.
